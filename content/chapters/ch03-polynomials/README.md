> **本章导读**：多项式是代数几何的核心主角！在后面的课程中我们会看到，**仿射空间本质上就是「多项式环的几何」**。这一章我们将从基础出发，为后续的代数几何学习打下坚实的多项式基础。

## 3.1 什么是多项式？

### 3.1.1 从一次多项式开始

回想我们中学学过的**一次函数**：

```
y = ax + b
```

这就是一个**多项式（Polynomial）**！在数学上，我们更一般地定义为：

> **多项式**是一个形如：
> 
> ```
> f(x) = a_n x^n + a_{n-1} x^{n-1} + ... + a_1 x + a_0
> ```
> 
> 的表达式，其中系数 a_i 是某个数域（ℚ, ℝ, ℂ）中的数。

### 3.1.2 多项式的系数和次数

```text
f(x) = 3x² + 2x + 1
        ↑  ↑  ↑  ↑
       a₂ a₁ a₀

- 系数：3, 2, 1
- 次数（Degree）：2（最高次幂）
- 记作：deg(f) = 2
```

```text
常数多项式：f(x) = 5
零多项式：f(x) = 0（特殊）
```

### 3.1.3 多项式的直观理解

想象多项式是一条**曲线**：

```text
        f(x) = x²
        │
    4   │        ●
        │      ●   ●
    3   │     ●     ●
        │    ●       ●
    2   │   ●         ●
        │  ●           ●
    1   │ ●             ●
        │●               ●
    0   ├─────────────────────
        0   1   2   3   4   x
```

不同的次数，曲线形态不同：
- **一次（线性）**：直线
- **二次**：抛物线
- **三次**：S形曲线

## 3.2 多项式环

### 3.2.1 环的定义

在深入多项式之前，我们需要理解**环（Ring）**的概念：

> **环**是一个集合 R，配备两种运算 + 和 ×，满足：
> 1. (R, +) 是阿贝尔群
> 2. 乘法有单位元 1
> 3. 乘法结合律
> 4. 分配律：a(b + c) = ab + ac

简单说：环就是「能加减乘」的集合。

### 3.2.2 多项式环

**多项式环**就是「多项式组成的环」：

```
K[x] = { a_n x^n + ... + a_0 | a_i ∈ K }
```

其中 K 是 ℚ, ℝ, ℂ 等数域。

```text
例如：
ℚ[x] = 有理系数多项式环
ℝ[x] = 实系数多项式环  
ℂ[x] = 复系数多项式环
```

### 3.2.3 环的运算

```text
f(x) = 2x² + 3x + 1
g(x) = x² - x + 4

加法：f + g = (2+1)x² + (3-1)x + (1+4)
     = 3x² + 2x + 5

乘法：f · g = (2x² + 3x + 1)(x² - x + 4)
     = 2x⁴ + (-2+3)x³ + (8-3+1)x² + (12-1)x + 4
     = 2x⁴ + x³ + 6x² + 11x + 4
```

## 3.3 多项式的根

### 3.3.1 根的定义

> 如果 f(a) = 0，则称 **a 是 f(x) 的根（Root）** 或 **零点（Zero）**。

```text
f(x) = x² - 4
f(2) = 2² - 4 = 0  →  2 是根
f(-2) = (-2)² - 4 = 0  →  -2 是根

所以 x² - 4 = (x-2)(x+2)
```

### 3.3.2 余数定理

> **余数定理**：用 x - a 除 f(x)，余数等于 f(a)。

```text
f(x) = x² - 3x + 2
f(1) = 1 - 3 + 2 = 0

所以 x - 1 是因子：f(x) = (x-1)(x-2)
```

### 3.3.3 代数基本定理

> **代数基本定理**：每个 n 次复数多项式恰好有 n 个复数根（计重数）。

```text
例子：
x² - 1 = 0  →  根：1, -1（2个根，次数=2）
x³ - 1 = 0  →  根：1, ω, ω²（3个根，次数=3）
```

## 3.4 最大公约数与欧几里得算法

### 3.4.1 整数的最大公约数

回想两个整数的**最大公约数（GCD）**：

```
gcd(a, b) = 能够整除 a 和 b 的最大正整数
```

**欧几里得算法**（辗转相除法）：

```
例如：求 gcd(48, 18)

48 = 18 × 2 + 12
18 = 12 × 1 + 6
12 = 6 × 2 + 0

所以 gcd(48, 18) = 6
```

### 3.4.2 多项式的最大公约数

同样的概念适用于多项式！

```
gcd(f(x), g(x)) = 能够同时整除 f 和 g 的次数最高的多项式
```

### 3.4.3 欧几里得算法（多项式版）

```text
求 gcd(x³ + 1, x² - 1)

x³ + 1 = (x² - 1) · (x) + (x + 1)
x² - 1 = (x + 1) · (x - 1) + 0

所以 gcd(x³ + 1, x² - 1) = x + 1
```

## 3.5 因式分解

### 3.5.1 唯一分解定理

> **唯一分解定理（UFD）**：在 ℚ[x], ℝ[x], ℂ[x] 中，每个多项式可以唯一地分解为不可约多项式的乘积。

```text
在 ℂ[x] 中：
x⁴ - 1 = (x-1)(x+1)(x-i)(x+i)

在 ℝ[x] 中：
x⁴ - 1 = (x-1)(x+1)(x²+1)

在 ℚ[x] 中：
x⁴ - 2 = (x² - √2)(x² + √2) 不可约？需要视情况而定
```

### 3.5.2 艾森斯坦判别法

如何判断一个多项式是否**不可约**？

> **艾森斯坦判别法**：如果存在素数 p，满足：
> - p 整除所有系数（除了首项）
> - p 不整除首项系数
> - p² 不整除常数项
> 
> 则该多项式在 ℚ[x] 中不可约。

```text
例子：f(x) = x³ + 2x + 6

取 p = 2：
- 2 整除 2（第二项系数）✓
- 2 整除 6（常数项）✓
- 2 不整除 1（首项系数）✓
- 2² = 4 不整除 6 ✓

所以 f(x) 在 ℚ[x] 中不可约！
```

## 3.6 多元多项式

### 3.6.1 二元多项式

到目前为止我们只讨论了一元多项式。代数几何中更重要的是**多元多项式**！

```text
f(x, y) = x² + xy + y²
g(x, y) = x³ - y³ + 2xy
```

**总次数**：各项次数之和的最大值

```text
f(x, y) = x² + xy + y²
         x² (次数=2)
         xy (次数=2)  
         y² (次数=2)
         
所以总次数 = 2
```

### 3.6.2 多元多项式环

```text
K[x, y] = 二元多项式环
K[x, y, z] = 三元多项式环
```

```text
例如：在 ℂ[x, y] 中
f(x, y) = x² + y² - 1

这就是单位圆的方程！
```

## 3.7 理想：多项式的「倍数」集合

### 3.7.1 理想的定义

> **理想（Ideal）** 是多项式环的一个特殊子集 I，满足：
> 1. 若 f, g ∈ I，则 f + g ∈ I
> 2. 若 f ∈ I，h ∈ K[x]，则 hf ∈ I

简单说：理想对**加法封闭**，对**乘法也封闭**（吸收任意多项式）。

### 3.7.2 主理想

> 由单个多项式生成的理想叫**主理想**：

```
⟨f⟩ = { f · h | h ∈ K[x] }
```

### 3.7.3 例子

```text
在 ℚ[x] 中：
⟨x²⟩ = { x² · h(x) | h(x) ∈ ℚ[x] }
      = { 0, x², 2x², x³, x⁴, ... }

⟨x² - 1⟩ = { (x² - 1) · h(x) | h(x) ∈ ℚ[x] }
```

> 💡 **重要**：在代数几何中，**理想对应着几何对象**！
> 
> - 理想 ⟨f₁, f₂, ..., fₖ⟩ 对应的几何是这些多项式的**公共零点集合**
> - 这就是**代数簇（Algebraic Variety）**的起点！
