# 第一章：集合与逻辑 —— 数学世界的基石

> **本章导读**：集合是现代数学最基本的语言，逻辑是推理的规则。这一章我们将从日常生活中的例子出发，轻松入门集合与逻辑，为后续所有章节打下基础。

## 1.1 什么是集合？

想象一下你有一个书包，里面放着数学书、语文书、英语书。我们可以说：书包里有「书的集合」。

在数学中，**集合（Set）** 就是「一堆东西放在一起」的整体概念。

### 1.1.1 集合的表示

```text
┌─────────────────────────────────────┐
│           集合 A = {1, 2, 3}        │
│                                     │
│      • 1   • 2   • 3               │
│         (元素们)                     │
└─────────────────────────────────────┘
```

**常见表示方法**：

1. **列举法**：把元素全部写出来
   ```
   A = {1, 2, 3, 4, 5}
   B = {苹果, 香蕉, 橙子}
   ```

2. **描述法**：用条件描述元素
   ```
   C = {x | x 是偶数}   // 读作：所有满足"x是偶数"的x
   D = {x | 0 < x < 10} // 0到10之间的数
   ```

### 1.1.2 一些重要的集合

| 符号 | 含义 | 例子 |
|------|------|------|
| ℕ | 自然数 | 1, 2, 3, ...（或 0, 1, 2, ...） |
| ℤ | 整数 | ..., -2, -1, 0, 1, 2, ... |
| ℚ | 有理数 | 可以写成分数的数 |
| ℝ | 实数 | 所有小数，包括π、√2 |
| ℂ | 复数 | a + bi 形式的数 |

```text
    ┌────────────────────────────────────┐
    │         ⊂  ⊂  ⊂  ⊂                │
    │    ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ ⊂ ℂ             │
    │    (子集关系：自然数是整数的子集)   │
    └────────────────────────────────────┘
```

## 1.2 集合的运算

### 1.2.1 并集 (Union)

把两个集合合并在一起：

```text
    A = {1, 2, 3}
    B = {3, 4, 5}
    
    A ∪ B = {1, 2, 3, 4, 5}
    
    图示：
    ┌───┐   ┌───┐
    │1 2│ ∪ │3 4│ = {1,2,3,4,5}
    │3  │   │5  │
    └───┘   └───┘
```

**符号**：∪（像字母U，代表 Union）

### 1.2.2 交集 (Intersection)

两个集合共同的部分：

```text
    A = {1, 2, 3}
    B = {3, 4, 5}
    
    A ∩ B = {3}
    
    图示：
    ┌───┐   ┌───┐
    │1 2│ ∩ │3 4│ = {3}
    │3  │   │5  │
    └───┘   └───┘
```

**符号**：∩（像倒过来的U，代表 Intersection）

### 1.2.3 差集 (Difference)

属于A但不属于B的部分：

```text
    A = {1, 2, 3}
    B = {3, 4, 5}
    
    A \ B = {1, 2}
    B \ A = {4, 5}
```

### 1.2.4 补集 (Complement)

相对于某个「全集」U：

```text
    U = {1, 2, 3, 4, 5}  (全集)
    A = {1, 2}
    
    A的补集 = U \ A = {3, 4, 5}
    
    记作：Aᶜ 或 U \ A
```

## 1.3 集合的关系：子集

### 1.3.1 子集的定义

如果A的每个元素都在B中，那么A是B的子集：

```
A ⊆ B  ⇔  对每个 x ∈ A，都有 x ∈ B
```

```text
    B = {1, 2, 3, 4, 5}
    A = {1, 2, 3}
    
    则 A ⊆ B
    
    图示：
    ┌───────────────┐
    │   B           │
    │  ┌─────────┐  │
    │  │   A     │  │
    │  └─────────┘  │
    └───────────────┘
```

### 1.3.2 真子集

如果A是B的子集，但A ≠ B，那么A是B的真子集：

```
A ⊂ B  ⇔  A ⊆ B 且 A ≠ B
```

### 1.3.3 两个重要的集合

- **空集**：没有任何元素的集合
  ```
  ∅ = {}  或  ∅
  ```
  空集是所有集合的子集：∅ ⊆ A（对任何集合A）

- **全集**：讨论补集时的参考集合，通常记作 U

## 1.4 逻辑基础

### 1.4.1 命题

**命题**是一个可以判断真假的陈述句。

| 是命题 | 不是命题 |
|--------|----------|
| 「2 + 2 = 4」 | 「你好吗？」 |
| 「3是偶数」 | 「请关门」 |
| 「π是无理数」 | 「明天会下雨」* |

*「明天会下雨」在明天到来前无法判断真假，但原则上可以判断，所以也可以视为命题。

### 1.4.2 逻辑联结词

#### 且 (And, ∧)

```
P ∧ Q 为真  ⇔  P为真 且 Q为真
```

| P | Q | P ∧ Q |
|---|---|-------|
| 真 | 真 | 真 |
| 真 | 假 | 假 |
| 假 | 真 | 假 |
| 假 | 假 | 假 |

#### 或 (Or, ∨)

```
P ∨ Q 为真  ⇔  P为真 或 Q为真（或两者都真）
```

| P | Q | P ∨ Q |
|---|---|-------|
| 真 | 真 | 真 |
| 真 | 假 | 真 |
| 假 | 真 | 真 |
| 假 | 假 | 假 |

#### 非 (Not, ¬)

```
¬P 为真  ⇔  P为假
```

| P | ¬P |
|---|----|
| 真 | 假 |
| 假 | 真 |

#### 蕴含 (Implies, ⇒)

```
P ⇒ Q 读作「如果P，那么Q」
P ⇒ Q 为假 只有一种情况：P为真且Q为假
```

| P | Q | P ⇒ Q |
|---|---|-------|
| 真 | 真 | 真 |
| 真 | 假 | 假 |
| 假 | 真 | 真 |
| 假 | 假 | 真 |

> 💡 **思考**：为什么「如果P为假，那么P⇒Q总是真」？
> 
> 想象你说：「如果太阳从西边出来，我就给你100块。」
> 太阳没有从西边出来（前提为假），所以我说的话不算撒谎——这在数学上视为「真」。

#### 等价 (Equivalent, ⇔)

```
P ⇔ Q  ⇔  (P ⇒ Q) 且 (Q ⇒ P)
```

读作「P当且仅当Q」或「P等价于Q」。

## 1.5 量词

### 1.5.1 全称量词 (∀)

「对所有的...」「对每一个...」

```
∀x ∈ ℕ, x² ≥ 0
读作：对所有自然数x，x的平方大于等于0
```

### 1.5.2 存在量词 (∃)

「存在...」「至少有一个...」

```
∃x ∈ ℕ, x + 5 = 10
读作：存在自然数x，使得x+5=10
```

### 1.5.3 否定

```
¬(∀x, P(x))  ⇔  ∃x, ¬P(x)
¬(∃x, P(x))  ⇔  ∀x, ¬P(x)
```

例子：
- 「所有人都喜欢数学」的否定是「存在一个人不喜欢数学」

## 1.6 习题

### 基础题

1. 用列举法表示：
   - {x | x是正方形且边长小于5}
   - {x ∈ ℕ | x是偶数且 x < 10}

2. 已知 A = {1, 2, 3}，B = {3, 4, 5}，求：
   - A ∪ B
   - A ∩ B
   - A \ B

3. 判断真假：
   - ∅ ⊆ {1, 2}
   - \{1\} ∈ \{\{1\}, \{2\}\}

### 进阶题

4. 证明：对于任意集合A，有 A ∩ ∅ = ∅

5. 化简：(A ∩ B) ∪ (A ∩ Bᶜ)

6. 用真值表验证：P ⇒ Q 与 ¬P ∨ Q 等价

### 挑战题

7. 描述以下集合：
   - {x ∈ ℤ | ∃y ∈ ℤ, x = 2y}（这是什麼？）
   - {x ∈ ℕ | ∀y ∈ ℕ, y > 0 ⇒ x ≥ y}（这又是什麼？）
