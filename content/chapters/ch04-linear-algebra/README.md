# 第四章：线性代数入门 —— 向量与空间

> **本章导读**：线性代数是现代数学的「通用语言」。无论是物理、工程还是代数几何，都离不开向量空间和线性变换。这一章我们将从直观出发，逐步深入，为理解后续的模层理论做准备。

---

## 4.1 向量：从一行一列开始

### 4.1.1 什么是向量？

**向量（Vector）** 就是「有方向和大小」的量。

```text
直观理解：
         ↑
         │   向量 v
    3    │  / 
    ↑    │ /
    │    │/
    0----+------→ 
    0    2   x

v = (2, 3)  从原点 (0,0) 指向 (2,3)
```

在数学上，我们把向量写成列的形式：

```
v = [2]
    [3]
```

### 4.1.2 向量的运算

**加法**：对应分量相加

```
[1]   [2]   [3]
[2] + [3] = [5]
```

**数乘**：每个分量都乘以常数

```
3 × [1] = [3]
      [2]   [6]
```

### 4.1.3 向量的几何意义

```text
加法的几何意义（平行四边形法则）：

    ● b
   /
  /
 ● a

  a + b

    ● b
   / \
  /   \
 ●-----●
 a   a+b
```

## 4.2 向量空间

### 4.2.1 定义

> **向量空间（Vector Space）** 是一个集合 V，配有：
> - 向量加法：V × V → V
> - 标量乘法：K × V → V（K 是 ℚ, ℝ, ℂ 等域）
> 
> 满足一系列公理（交换律、结合律、单位元、逆元等）。

```text
简单说：向量空间就是一个「可以做线性组合」的集合。
```

### 4.2.2 例子

| 向量空间 | 描述 |
|----------|------|
| ℝⁿ | n 维实向量 |
| ℝ[x] | 实系数多项式 |
| ℝ^{m×n} | m×n 实矩阵 |
| C[0,1] | [0,1] 上连续函数 |

```text
重要例子：ℝ²（二维平面）

        ↑
    y   │
        │        ● v = (3,2)
        │       
        │    ● w = (1,1)
        │
        +------------→ x
        
任意向量可以写成 (x, y) = x·(1,0) + y·(0,1)
```

### 4.2.3 子空间

> **子空间（Subspace）** 是向量空间的子集，且本身也是向量空间。

```text
在 ℝ³ 中：
- 通过原点的直线是子空间（1维）
- 通过原点的平面是子空间（2维）
- ℝ³ 本身是子空间（3维）
- {0} 是零空间（0维）
```

---

## 4.3 基与维数

### 4.3.1 线性组合

> **线性组合**是向量的「加权和」：

```
v = a₁v₁ + a₂v₂ + ... + aₙvₙ
```

### 4.3.2 线性相关与无关

> **线性无关**：没有非平凡线性组合等于零向量
> 
> 即：若 a₁v₁ + ... + aₙvₙ = 0，则必有 a₁ = ... = aₙ = 0

```text
例子：在 ℝ² 中
v₁ = (1, 0), v₂ = (0, 1) 线性无关
v₁ = (1, 0), v₂ = (2, 0) 线性相关（因为 2v₁ - v₂ = 0）
```

### 4.3.3 基（Basis）

> **基**是一组线性无关的向量，它们可以**张成**整个空间。

```text
ℝ² 的标准基：
e₁ = (1, 0)
e₂ = (0, 1)

任何向量 (a, b) = a·e₁ + b·e₂
```

### 4.3.4 维数

> **维数（Dimension）** 是基中向量的个数。

```text
dim ℝⁿ = n
dim ℝ[x] = ∞（无穷维！）
dim ℝ^{2×3} = 6
```

---

## 4.4 线性变换

### 4.4.1 定义

> **线性变换（Linear Transformation）** T: V → W 满足：
> 1. T(v + w) = T(v) + T(w)
> 2. T(av) = aT(v)

```text
简单说：线性变换「保持加法」和「保持数乘」。

例子：旋转、缩放、投影都是线性变换！
```

### 4.4.2 矩阵表示

每个线性变换都可以用**矩阵**表示：

```text
T: ℝ² → ℝ²

T(x, y) = (2x, 3y)  ← 这是一个「缩放变换」

矩阵表示：
[2  0]
[0  3]
```

### 4.4.3 例子

```text
1. 恒等变换：I(v) = v     → 矩阵 [1 0; 0 1]
2. 零变换：0(v) = 0      → 矩阵 [0 0; 0 0]
3. 旋转（90°）：        → 矩阵 [0 -1; 1 0]
4. 投影到 x 轴：        → 矩阵 [1 0; 0 0]
```

---

## 4.5 行列式

### 4.5.1 直观理解

> **行列式（Determinant）** 是方阵的一个「数值」，它告诉我们线性变换的「体积缩放因子」。

```text
2×2 矩阵的行列式：

|a  b|     ad - bc
|c  d| →

几何意义：平行四边形的面积

    (a,b)    ●
          ↙
        ↙ area = |ad - bc|
      ↙
(c,d) ●
```

### 4.5.2 计算公式

```text
2×2 行列式：
| a  b | = ad - bc
| c  d |

3×3 行列式（展开公式）：
| a b c |
| d e f | = a(ei - fh) - b(di - fg) + c(dh - eg)
| g h i |
```

### 4.5.3 行列式的性质

```text
1. |AB| = |A|·|B|
2. |A| = 0 ⇔ A 不可逆（非满秩）
3. 交换两行，行列式变号
4. 某行乘以 k，行列式也乘以 k
```

## 4.6 特征值与特征向量

### 4.5.1 定义

> **特征向量（Eigenvector）** 在变换后方向不变，只是伸缩了：
> 
> Av = λv
> 
> 其中 λ 叫做**特征值（Eigenvalue）**。

```text
例子：缩放变换

A = [2 0]
    [0 3]

v₁ = (1, 0) → Av₁ = (2, 0) = 2v₁  → λ₁ = 2
v₂ = (0, 1) → Av₂ = (0, 3) = 3v₂  → λ₂ = 3
```

### 4.5.2 特征方程

> 求特征值，就是解 **特征方程**：
> 
> det(A - λI) = 0

```text
例子：A = [2 1]
         [1 2]

特征方程：
|2-λ   1  | = (2-λ)² - 1 = λ² - 4λ + 3 = 0
| 1   2-λ|

解：λ = 1 或 λ = 3
```

### 4.5.3 对角化

> 如果有 n 个线性无关的特征向量，则矩阵可以对角化：
> 
> A = PDP⁻¹

```text
这在代数几何中用于研究线性层的截面！
```

## 4.7 秩与核

### 4.7.1 秩（Rank）

> **秩（Rank）** 是线性变换的「有效维度」：
> - **列秩**：列向量张成空间的维数
> - **行秩**：行向量张成空间的维数
> - 两者相等！

### 4.7.2 核（Kernel）

> **核（Kernel）** 是被映射到零的向量集合：
> 
> Ker(T) = {v ∈ V | T(v) = 0}

### 4.7.3 维数定理

> **维数定理**：
> 
> dim V = dim Ker(T) + dim Im(T)

```text
这是线性代数最重要的定理之一！
```

## 4.8 线性代数与代数几何

> 💡 **重要联系**：线性代数是学习代数几何的必要基础！
> 
> 在代数几何中：
> - **向量空间** → **模（Module）**
> - **线性变换** → **态射（Morphism）**
> - **特征值** → **层的上同调**
> - **行列式** → **行列式层（Determinant Line Bundle）**

> 不要担心这些术语——以后我们都会详细介绍！现在只需要有个印象就可以了。
